支付问题

公众号网页拉起的支付请求在小程序是拉不起的，小程序内有自已支付 API。所以在公众号网页下的单到支付这部把生成的订单号通过 wx.miniProgram.navigateTo 把订单号传到小程序。后面的跟公众号支付类似 参考。小程序支付 API wx.requestPayment(OBJECT) 发起微信支付
