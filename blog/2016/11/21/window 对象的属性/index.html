浏览器 window 对象

2016/11/21

本文于1100天之前发表，文中内容可能已经过时。

今天收集了下关于浏览器中的 window 对象的一些相关信息。

window 对象的属性

  1. window，name 属性

    window.window属性指向是自身。

    window.name属性用于设置当前浏览器窗口的名字。

  2. location 属性

    window.location返回一个Location对象，用于获取窗口当前的URL信息。

  1. closed, opener 属性

    window.closed属性返回一个布尔值，表示窗口是否关闭。

    window.opener属性反回打开当前窗口的父窗口。如果当前窗口没有父窗口，则返回 null。

  2. length 属性

    window.length属性返回当前网页包含的框架总数。如果当前网页不包含 frame 和 iframe 元素，就返回 0。

  3. screenX, screenY 属性

    window.screenX和window.screenY属性，返回浏览器窗口左上角相对于当前屏幕左上角的水平距离和垂直距离，单位为像素。

  4. innerWidth, innerHeight 属性

    window.innerWidth和window.innerHeight属性返回网页当前窗口中可见部分的宽度和高度，单位像素。当页面放大时（如页面从100%大小放大到200%时）window.innerWidth和window.innerHeight返回的宽度和高度是100%页面时的大小的一半。

  5. scrollX, scrollY 属性

    window.scrollX和window.scrollY属性返回页面的水平滚动距离和垂直滚动距离，单位为像素。
