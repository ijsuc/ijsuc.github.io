ref回调属性

React
ref属性需要一个回调函数，该主件被安装或卸载后立即回调将被执行

ref属是在 HTML 元素上使用的，｀ref｀回调接收底层DOM元素作为其参数。例如下面的代码：

<div ref={ el => this.myDiv = el }>示例代码</div> el参数是<div>示例代码</div>, this.myDiv是该主件上的一个自定义属性。
